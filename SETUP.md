# Ondiary セットアップガイド

## 🚀 すぐに始める

### ステップ1: 開発サーバーを起動

新しいコマンドプロンプト（またはPowerShell）を開いて：

```cmd
cd C:\Users\MM199\Documents\ondiary
npm run dev
```

### ステップ2: ブラウザで開く

http://localhost:5173/ にアクセス

### ステップ3: マイク許可

ブラウザがマイクへのアクセス許可を求めたら「許可」をクリック

### ステップ4: 日記を書く

1. 「日記を始める」ボタンをクリック
2. 質問が音声で読み上げられます
3. マイクボタン🎤をタップして回答
4. 15問すべてに回答すると日記が完成！

---

## 📋 APIキー設定（任意）

より洗練された日記を生成したい場合：

### Claude APIを使用

1. `src/utils/aiApi.js` を開く
2. 12行目を編集：

```javascript
const API_KEY = 'あなたのClaudeAPIキーをここに'
```

3. APIキーは https://console.anthropic.com/ で取得

## ⚠️ トラブルシューティング

### 音声が認識されない
- Chromeまたは Edgeブラウザを使用
- マイク許可を確認
- ページを再読み込み

### 日記が生成されない
- APIキーが正しく設定されているか確認
- APIキーなしでもモック日記が生成されます

### サーバーが起動しない
- Node.jsがインストールされているか確認: `node --version`
- 依存関係を再インストール: `npm install`
